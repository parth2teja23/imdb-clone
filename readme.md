## To run the project:
1. clone the repository
2. Run the following commands:
```
cd imdb 
python manage.py runserver
```
3. To run TailwindCSS, add another tyerminal and run follwing:
```
cd imdb # ignore if alrealy in imdb
python manage.py tailwind start
```
